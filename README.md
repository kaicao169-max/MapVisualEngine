**Project Title: Geospatial Data Visualization Engine(Python/Folium)**

**Qverview:** A high-performance, responsive mapping solution designed for environmental monitoring.

**Key Features:**

* **Adaptive UI:** Fully responsive design(Desktop & Mobile) using viewport-aware CSS logic.
* **Modular Components:** Custom-built card popups, markers and legends decoupled from main data processing.

* **High Scalability:** JSON-driven configuration for rapid map style deployment.
